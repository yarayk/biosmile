Мобильное приложение для поддержки реабилитации детей с нарушениями речи и мимики после инсульта (4–10 лет), где на Flutter/Dart реализован кроссплатформенный интерфейс с игровыми и адаптивными упражнениями, локальной обработкой мимики с помощью MediaPipe, хранением данных и аутентификацией через Supabase (PostgreSQL), а также бэкенд-логикой на Supabase (BaaS), включающей API-доступ, управление контентом и синхронизацию, что обеспечило интерактивный UX, мониторинг прогресса и возможность масштабирования решения.
